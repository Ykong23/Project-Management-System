# WordPress Project Management System
Spearheaded end-to-end development of company project management system. Implemented full system modules (dashboard/leave/claims/project tracking), enhanced UI/UX, and optimized data presentation for improved user experience.

## Overview
The **Project Management System** is a tailored solution developed for **Apploqic Technologies PLT** to facilitate efficient project management and streamline internal processes. Created using **WordPress** and hosted on **CPanel**, this system serves as a vital tool for the company's employees, enabling them to manage projects, tasks, and administrative functions effectively.

## Purpose
The primary objective of this project is to provide a cost-effective management system that enhances collaboration among employees, simplifies project tracking, and improves overall operational efficiency. By leveraging the skills of a team of interns, including myself and three other students, we aimed to create a robust platform that meets the specific needs of the organization.

## Features
### User Role
The system incorporates two distinct roles: **Admin** and **User**
* User Side:
  * Dashboard: A comprehensive overview is provided for all ongoing projects and tasks. The user dashboard also displays an annual leave summary, including both the Used Days and Remaining Days. In addition, users can view their total Used Claim Hours directly from the dashboard.
  * View Projects and Tasks: Easy to access project details and claim task assginments.
  * Leave Application: Users can submit annual and medical leave requests seamlessly.
  * Claim Hours: A straightforwards process for users to log hours realated to their leave.
  * Profile Management: Users can update thier personal information.

* Admin Side:
  * Dashboard: A comprehensive overview is provided for all ongoing projects and tasks. The user dashboard also displays an annual leave summary, including both the Used Days and Remaining Days. In addition, users can view their total Used Claim Hours directly from the dashboard.
  * View Projects: Admin can monitor project statuses and progress.
  * Manage Leaves: The ability to review and approve leave applications.
  * Manage User: Admins can add, remove, or modify user accounts as necessary.

## Project Highlights
- **Cost-Effective Solution**：The system is provided free of charge to all Apploqic employees, significantly reducing operational costs.
- **Enhanced Project Management**：The platform simplifies the management of project worklogs, making it easier for teams to collaborate across departments.
- **Improved Communication**：Employees can easily track project progress and communicate their leave applications to management.
